BMI Calculator
Description
The "BMI Calculator" is a web application that helps users calculate their Body Mass Index (BMI). It provides an easy and quick way to assess whether a person has a healthy body weight for a given height.

Features
Calculate BMI based on height and weight inputs.
Provide BMI categories (Underweight, Normal weight, Overweight, Obesity) based on the calculated BMI.
User-friendly interface with clear input fields.
Responsive design for both mobile and desktop use.
Technologies Used
HTML
CSS
JavaScript
